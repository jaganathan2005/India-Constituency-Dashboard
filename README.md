📊 India Constituency Dashboard 🇮🇳
🔍 Overview

This project analyzes state-wise constituency seat distribution across India, covering 4126 constituencies.

📌 Features
*State-wise analysis
*Party-wise breakdown (NDA, INDIA, Others)
*Interactive dashboard
*Geographic visualization

🛠️ Tools Used
Power BI
Data Cleaning & Transformation

💡 Key Insight

Regional political dynamics vary significantly across states despite national trends.
